Substitution Cipher

How to use

1. Enter Substitution Alphabet: Input a 26-character long substitution alphabet in the "Enter Substitution Alphabet" text box. Ensure it contains all uppercase letters, no duplicates, and no missing letters (Example - Enter Substitution Alphabet: QWERTYUIOPLKJHGFDSAZXCVBNM).

2. Enter Plaintext: Type or paste the text you want to encrypt in the "Enter your text" textarea (Example - Enter your text: HELLO).

3. See Translated Text: The translated text will automatically appear in the "Translated text" section as you type in the alphabet and plaintext fields (Example - Result: ITSSG).

4. Reset: Click the "Reset" button to clear the input fields and the translated text.

Advanced Encryption Standard (AES)

How to use

Encrypt 1. Enter a 32-character key in the "Enter Key (32 characters)" input field (Example - Key: 12345678901234567890123456789012).

Encrypt 2. Enter the plaintext in the "Enter plaintext" textarea (Example - Plaintext: Hello World!).

Encrypt 3. Click the "Encrypt" button to encrypt the plaintext. The IV and encrypted data will be displayed in the "Output" section (Example - IV: 15,47,61,... Encrypted: 50,160,138,...).

Decrypt 1. Ensure the same 32-character key is used.

Decrypt 2. Click the "Decrypt" button to decrypt the previously encrypted data. The decrypted plaintext will be displayed in the "Output" section (Example - Decrypted: Hello World!).

Triple (3DES)

How to use

1. Enter a 24-character key in the 'Key' input field (Example - Key: 369876365016324577956874). The key is required for both encryption and decryption.

2. Enter the message you want to encrypt in the 'Message' input field (Example - Plaintext: Hello World!).

3. Click 'Encrypt' to encrypt the message using Triple DES (Example - Encrypted: U2FsdGVkX1+lE9sYNlZznGsljZzFOfNGcd6jCBlaShk=).

4. The encrypted message will be displayed. Click 'Decrypt' to decrypt it back to the original message using the same key (Example - Decrypted: Hello World!)
